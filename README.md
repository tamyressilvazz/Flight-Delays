# ✈️ Flight Delays and Air Travel Recovery Analysis

> Academic project developed during the **Data Science Essentials with Python** course from the Cisco Networking Academy.

## 📖 Overview

This project analyzes flight delays and air travel activity in the United States using datasets related to commercial flights, airport security checkpoint screenings, and daily airline passenger volumes.

The goal is to explore patterns in flight delays, understand air transportation trends, and investigate the impact of major events on passenger traffic over time.

Using Python and data analysis techniques, the project combines multiple datasets to uncover insights about airline operations and travel demand.

---

## 🎯 Objectives

* Analyze flight delay patterns in the United States.
* Explore trends in daily air passenger volumes.
* Investigate changes in airport traffic over time.
* Compare flight activity before and after major disruptions.
* Apply data science techniques to real-world transportation datasets.

---

## 📊 Datasets

### Flights Dataset

Contains information related to commercial flights and delays.

**Main Attributes**

| Column             | Description                     |
| ------------------ | ------------------------------- |
| Flight Information | Flight operational data         |
| Delay Metrics      | Departure and arrival delays    |
| Airline Data       | Airline identifiers             |
| Airport Data       | Origin and destination airports |

The dataset is used to investigate delay patterns and operational performance.

---

### US Daily Passengers Dataset

Contains daily passenger volume information for air travel in the United States.

**Main Attributes**

| Column          | Description                   |
| --------------- | ----------------------------- |
| Date            | Observation date              |
| Passenger Count | Number of screened passengers |

This dataset helps measure air travel demand over time.

---

### TSA Checkpoint Departures Dataset

Contains airport checkpoint screening information.

**Main Attributes**

| Column             | Description                   |
| ------------------ | ----------------------------- |
| Date               | Screening date                |
| Traveler Volume    | Number of travelers processed |
| Comparison Metrics | Historical comparison values  |

This dataset provides additional context regarding airport activity and passenger movement.

## 🔬 Analyses Performed

### Flight Delay Analysis

The project explores:

### Passenger Traffic Analysis

Passenger data is analyzed to identify:

### Airport Activity Trends

Using checkpoint screening data, the project investigates:

* Daily traveler volumes;
